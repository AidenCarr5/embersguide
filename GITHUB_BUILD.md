# Build the Mac DMG with GitHub Actions

1. Create a new GitHub repository.
2. Upload or push this folder to the repository.
3. Open the repository on GitHub.
4. Go to **Actions**.
5. Choose **Build Mac DMG**.
6. Click **Run workflow**.
7. When the run finishes, download the `Embers-Tracker-macOS-DMG` artifact.

The DMG is unsigned unless you add Apple Developer signing/notarization later.
